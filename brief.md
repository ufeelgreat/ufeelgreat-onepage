## Guide de style

### Palette de couleurs
| Rôle | Valeur |
|------|--------|
| Fond principal | `#000000` |
| Accent gradient (Hero, Footer) | `linear-gradient(#FF4D4D → #FFB347)` |
| Cartes / conteneurs | `#121212` |
| Bordures cartes | gris très fin |
| Texte principal | `#FFFFFF` |
| Texte secondaire | `#A0A0A0` |

### Typographie
- Famille : Plus Jakarta Sans (Google Fonts)
- Titres : Extra-Bold (800)
- Corps : Regular (400), interlignage aéré (~1.6–1.8)

### Composants
- **Border-radius cartes** : 24px–32px
- **Toggle Switch** : bascule entre deux vues (Production Audiovisuelle / Management & Opérations)
- **Scroll Reveal** : Slide-up + Fade-in via Intersection Observer

### Responsive
- Mobile-first
- Desktop : grille 2–3 colonnes, larges marges
- Mobile : empilement vertical, 1 colonne, boutons optimisés pouce
